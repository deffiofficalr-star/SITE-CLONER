# 🌐 SITE-CLONER

**Копирование сайтов в один клик.**

---

## 📌 Что это?

SITE-CLONER — это программа, которая скачивает полную копию сайта: HTML, CSS, JS, картинки, шрифты. Сохраняет в папку, исправляет ссылки для локального открытия.

---

## 🚀 Быстрый старт

```bash
git clone https://github.com/deffiofficalr-star/SITE-CLONER.git
cd SITE-CLONER
pip install -r requirements.txt
python main.py
```

---

📁 Структура

```
site_clone/
├── index.html
├── images/
├── css/
├── js/
└── fonts/
```
